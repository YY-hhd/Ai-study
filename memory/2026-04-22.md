## 🐛 Bug 修复

### 1. Message.setDefaults 错误

**错误**: `element_ui__WEBPACK_IMPORTED_MODULE_4__.Message.setDefaults is not a function`  
**原因**: Element UI 的 Message 组件不支持 setDefaults 方法  
**修复**: 移除 main.js 中的 setDefaults 调用，改为组件内单独配置  
**时间**: 2 分钟定位并修复

### 2. 页面空白问题

**原因**: main.js 语法错误导致编译失败  
**解决**: 移除不支持的 API 调用  
**验证**: 页面正常显示

### 3. 导航返回功能

**需求**: 从子页面返回首页  
**实现**: 
- Home.vue 添加返回按钮（非首页时显示）
- Users.vue 和 Data.vue 添加返回首页按钮
- 使用 router.push('/') 跳转

# Atropos RL 环境

**功能**: 强化学习环境，用于训练 AI 助手决策策略

---

## 📋 核心概念

### RL 框架

```
状态 (State) → 策略 (Policy) → 动作 (Action)
    ↓                              ↓
奖励 (Reward) ← 环境 (Environment)
```

### 状态空间

- 对话上下文
- 用户情绪
- 任务复杂度
- 历史成功率

### 动作空间

- 选择工具
- 选择技能
- 回复风格
- 是否寻求帮助

### 奖励信号

- 用户满意度 (+1 ~ +5)
- 任务完成 (+10)
- 时间效率 (-0.1/ms)
- 错误惩罚 (-5)

---

## 🔄 训练流程

```
收集轨迹 → 计算回报 → 更新策略 → 评估效果
```
